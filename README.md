# 🇰🇿 KAZGEOMINER

Информационная система для управления доступом к геологическим данным по графиту в Казахстане.

## 🔥 Как запустить проект

Для быстрого запуска и фронтенда, и бэкенда одной командой:

1. Откройте терминал в корне проекта.
2. Сделайте скрипт исполняемым (только в первый раз):
   ```bash
   chmod +x start.sh
   ```
3. Запустите его:
   ```bash
   ./start.sh
   ```

## 🛠 Технологии

- **Frontend**: HTML5, Vanilla CSS, JavaScript (Lucide Icons, Animate.js)
- **Backend**: FastAPI (Python 3.10+), SQLAlchemy
- **Database**: SQLite
- **Security**: Passlib (BCrypt) для хеширования паролей
